Recent Updates
2/11/2026
  • Added a new B-Size template: X or Y. This lets you specify two different prices with different conditions. Usually used for Stock-Up Savings deals, but can be used for other purposes such as two different sizes of items. This template is still in progress so you can only use the regular dollars and cents price type for now.
  • Added an early version of a Blank C size template. This is still in testing so may not be aligned properly just yet.
8/21/2025
  • Added a Recent Signs button. Your signs will now be saved to this menu once you hit the Print button on the page. (You don't have to actually print it for it to save)
6/30/2025
  • You can now change the YOU SAVE text to say SAVE AT LEAST by specifying a range of prices (such as 4.99-6.99) (Only on B and C regular signs currently)
